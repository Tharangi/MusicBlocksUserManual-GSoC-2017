Auxillary Toolbar

The Auxillary toolbar is used to manage projects of music blocks. It includes loading, saving, design tools and visibility management

Load samples from server. This opens up a new page to load projects.
Load projects from local machine
Save project (in .tb format).
Save sheet music. Project will be saved in .ly format, which can be opened by Lilypond.
Paste blocks from the clipboard. (This button is highlighted only when there are blocks available on the clipboard to paste.)
To copy blocks(or stack of blocks), long press on the block, then the past button will be highlighted.
Show or hide a Cartesian-coordinate grid.
Show or hide a polar-coordinate grid.

Open utility panel to control functions in the workspace.

Decreases the block size.

Increases the block size.

Shows project stats including the usage of blocks of each palette.

Loads supported plugins from local machine.

Enables/disables scrolling

Remove all blocks.
Restore blocks from the trash.
