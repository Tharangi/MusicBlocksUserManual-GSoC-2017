Main Toolbar

The Main toolbar is used to run programs, erase the screen, and hide the palettes and blocks.


Run the blocks in normal speed.

When long pressed the run button: Run the blocks slowly

When extra long pressed the run button: Run the music slowly

Run the blocks slowly. When running slowly, the values of parameter boxes are shown as an additional debugging aid.
Run the blocks step by step (one block is executed per turtle per click).
Stop running the current project.
Clear the screen and return the turtles to their initial positions in the center of the screen.
Hide or show the block palettes.
Expand or collapse stacks of blocks (start and action stacks).
Re-arrange the blocks workspace - relative to the top left corner of the workspace.

When home pressed the button changes to: after re-arranging the workspace

Show the help messages.
