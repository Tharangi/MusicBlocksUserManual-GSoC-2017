Palette Toolbar

The Palette toolbar contains all the blocks needed to create music and graphics with Music Blocks. These blocks are arranged into various categories based on the intended functionalities and placed on the left side of the workspace.


Following shows what are the groups and how they are categorised:

  • Palettes for Music
    • Tone Palette
    • Intervals Palette
    • Drums Palette
    • Widgets Palette
  • Palettes for Graphics
    • Mouse Palette
    • Pen Palette
  • Palettes for Programming
    • Flow Palette
    • Action Palette
    • Boxes Palette
    • Number Palette
    • Boolean Palette
    • Sensors Palette
    • Heap Palette
  • Extra Widgets
    • Medium Palette
    • Extras Palette
