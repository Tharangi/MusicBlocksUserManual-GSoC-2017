Action Blocks



Action block contains a sequence of actions that will only be executed when the block is referred to by something else, such as a start block. This is useful in orchestrating more complex programs of music.
Start Block is a chunk that will automatically be executed once the start button is pressed. This is where most of programs will start. There can be more than one start block for a project. When the run button is pressed. all start blocks will start executing (unless otherwise special instruction sequences are specified.
