Components of Music Blocks

Music Blocks software consists of two main components.

  1. Workspace
  2. Toolbars
Arrangements of these two components are shown in quick start page.

1. Workspace

This is where the programming is done. You can drag and drop various components on Music Blocks, arrange them in various ways, create music and animations.

2. Toolbars

There are three toolbars provided in Music Blocks

  • Main toolbar is for run/debug purposes which is on top of the workspace.
  • Auxiliary toolbar is for managing projects and to control the workspace.
  • Palette toolbar contains all building blocks for creating music and animations.

Each of these toolbars contain various other components grouped under their specific functionalities.
